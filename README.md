## Proyectos destacados

### HVAC Predictive Maintenance & Energy Efficiency
Proyecto de análisis y Machine Learning para detectar riesgo de falla en equipos HVAC usando lecturas operativas como Delta T, presiones, amperaje, vibración, horas de operación y consumo energético.

### Facilities Maintenance KPI Analysis
Proyecto de análisis de órdenes de trabajo de mantenimiento para crear KPIs operativos, evaluar cumplimiento de SLA, identificar áreas con mayor costo y proponer acciones de mejora.

### Customer Churn Prediction
Proyecto de Machine Learning enfocado en predecir abandono de clientes bancarios utilizando clasificación supervisada, tratamiento de desbalance de clases y evaluación con F1-score y AUC-ROC.
